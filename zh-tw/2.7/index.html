Conan 2 - C 和 C++ 軟體包管理器文件¶

歡迎！這是 Conan 的使用者文件，Conan 是一個開源、去中心化的 C/C++ 軟體包管理器，支援所有平台以及所有建置系統與編譯器。其他相關資源：

  • Conan 主頁。專案的進入點，包含文件、網誌、社群、下載、發行郵件列表等連結。

  • Github 專案與議題追蹤器。這是主要的支援管道，請在此處提交問題、臭蟲回報和功能請求。
